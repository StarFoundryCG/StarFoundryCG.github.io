*Prior industry or academic experience is not required, yet most successful applicants have been introduced to programming by writing some code independently, or by working through courses on e-learning platforms such as CodeSchool or Codeacademy. This is not a "0–60" course, this is a "20–120" course. If you haven't written any code when you apply, we will ask you to do so during the admissions process.

Curious about joining The Star Foundry

Our interactive challenge will guide you through a quick coding test and collect basic information. If you pass, you'll have a coding session with an instructor - over Skype - who will teach you a few tricks, tell you more about the program, and assess whether the program is a good fit for you.
